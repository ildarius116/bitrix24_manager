## Что сделано по подзадачам

### 5_01 — Идемпотентность (журнал обработанных дней)

Новый модуль `src/journal.py` — локальный журнал `.runtime/processed.json` с TTL:
- `load_journal` / `is_processed` / `mark_processed` — чтение, проверка и запись с очисткой
  просроченных записей по TTL.
- Атомарная запись: сначала во временный файл `processed.json.tmp`, затем `os.replace`; при
  исключении `.tmp` очищается — журнал никогда не остаётся в повреждённом состоянии.
- Новый ключ `config.yaml`: `runtime.journal_ttl_days: 7`; геттер `Config.journal_ttl_sec`
  с защитой от нечислового значения.
- **Интеграция в `run_fill`:** в боевом режиме день, уже отмеченный в журнале в пределах TTL,
  получает статус `skipped` без обращения к порталу; после успешной боевой записи (`filled`)
  вызывается `mark_processed` (обёрнут в `try/except` — потеря журнала не роняет пайплайн).
- В `--dry-run` журнал не читается и не пишется.
- Журнал — дополнительный слой поверх главного гарда `_reread_guard` (перепроверка 4-дневного
  окна и пустоты перед каждой реальной записью).

сводная таблица статусов были полностью реализованы в Фазах 1 и 4. Новых правок в коде не

### 5_03 — Логирование и аудит REST

`src/b24.py` расширен единой аудируемой точкой вокруг `client.call`:
- Аудит охватывает `item_list_all` и `batch`.
- Путь к файлу: геттер `get_audit_file_path(config)` → `config.env.audit_file` →
  `B24_AUDIT_FILE` из `.env` → дефолт `.runtime/bitrix24_audit.jsonl`.
- В каждую строку аудита пишутся: `method`, `param_keys` (только **имена** ключей, без
  значений), `status`, `duration_ms`, `tenant` (domain). Значения параметров и код вебхука
  в аудит не попадают.
- Запись в аудит-файл обёрнута в `try/except` — ошибка журналирования не роняет REST-вызов.
- Структурный лог и маскирование секрета вебхука в выводе реализованы ещё в Фазе 1 и
  сохранены.
- Follow-up (не блокер): аудит включён по умолчанию и растёт без ротации — при накоплении
  больших объёмов рекомендуется настроить ротацию `.runtime/bitrix24_audit.jsonl` (например,
  `logrotate` или периодическая очистка вручную).
