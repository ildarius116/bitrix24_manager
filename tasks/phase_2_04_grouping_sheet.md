## Цель
Дополнительный лист с группировкой содержимого «Работы/задачи за день» по «Описанию задачи».

## Объём работ
- Лист «Группировка».
- Ключ группы — `ufCrm48_1744239302` (Описание задачи); пустое → «(без описания)».
- Агрегаты: кол-во записей, сумма часов (`ufCrm48_1742996959`), список дат.
- Сортировка по описанию (или по сумме часов — согласовать).

## Критерии приёмки (DoD)
- Каждая уникальная «Описание задачи» — одна строка с верными агрегатами.
- Сумма часов по группам сходится с основным листом.
