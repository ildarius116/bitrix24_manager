# phase_delayed_1_03 — Автозапуск по расписанию (отложено)

**Статус:** DELAYED

## Идея
Регулярный автозапуск `fill --no-interaction` (например, ежедневно в рабочее время), чтобы
заполнять «Работы/задачи за день» без ручного запуска. Планировщик Windows / cron / сервис.

## Когда брать
После стабилизации Фаз 1–5 и явного согласия заказчика на автоматические записи в прод.
Требует: безопасные дефолты, идемпотентность (phase_5_01), мониторинг ошибок.
